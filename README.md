# Free Lossless Audio Recorder Light

Lossless Music Recorder is a lightweight desktop app for recording music from an embedded WebView (Deezer, Qobut, TIDAL) or system audio using WASAPI loopback.
It saves recordings as FLAC files, reads track metadata automatically, supports auto-splitting on song changes, and embeds titles in the format Artist - Title.

There will be a paid Version where you can stream and record multiple tracks at the same time.
